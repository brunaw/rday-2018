Uma das áreas que vêm tomando importância em computação e estatística  é a de
Music Information Retrieval. Este campo pretende ampliar a compreensão e utilidade dos 
dados de música através de pesquisa, desenvolvimento e aplicação de ferramentas 
computacionais, combinadas com teoria e conceitos musicais.  É responsável pela criação de diferentes mecanismos de busca e recomendação para música, geração automática de composições,  inferência utilizando áudio ou representação simbólica de sons. Este trabalho tem o objetivo de apresentar os principais pacotes usados para a extração e utilização de dados musicais no R atualmente: o pacote music21, que porta a biblioteca em python de musicologia, de mesmo nome, para o R;  o  pacote choRds, que realiza a raspagem de acordes e possibilita a engenharia de covariáveis dos dados obtidos;  o pacote Rspotify, que faz a conexão do R com a API do maior software de streaming de música da atualidade, o Spotify; e o pacote tuneR, que faz a leitura de arquivos de aúdio em diferentes formatos.  Cada pacote tem suas espeficidades, que serão exploradas durante a apresentação, e trazem diferentes níveis de informação sobre as mesmas músicas. Por fim, pretende-se mostrar uma análise prática, envolvendo desde a extração até a análise dos dados usando estes pacotes, de forma a demonstrar como eles podem ser integrados para a obtenção de resultados mais interessantes.
